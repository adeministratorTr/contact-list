# Getting Started with Create React App

This project was bootstrapped with create react app and redux.

You can run project with `yarn` and `yarn start`

`yarn test` runs tests implemented with jest

`yarn test:cy` runs cypress tests

Live version available: https://floating-sands-82655.herokuapp.com/

### E2E Tests

Tip 1: before run end to end tests, first visit website. Since its heroku free domain, sometimes it will be a problem.

Tip 2: You can cypress base url and run it on your local too. Ensure you run server locally before run cypress tests.
